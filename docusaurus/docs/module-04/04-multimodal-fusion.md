# Multimodal Fusion

Fusing vision and language embeddings allows the robot to ground instructions into perceived objects.

## Learning Objectives
- Align image features with text prompts  
- Build cross-modal attention modules  

## Key Takeaways
- Fusion makes instructions actionable
