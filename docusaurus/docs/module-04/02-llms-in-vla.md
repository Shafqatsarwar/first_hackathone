# Large Language Models for Robotics

LLMs provide flexible instruction understanding, enabling natural queries, dialog management, and plan generation for robots.

## Learning Objectives
- Evaluate LLM capabilities and limits  
- Prompt for safe robot directives  

## Key Takeaways
- LLMs are reasoning partners, not controllers
