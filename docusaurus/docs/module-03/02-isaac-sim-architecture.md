# NVIDIA Isaac Sim Architecture

Isaac Sim provides photorealistic rendering, GPU-accelerated physics, and tight ROS 2 integration so AI brains can be built and validated on high-fidelity simulations.

## Learning Objectives
- Understand Isaac Sim structure  
- Integrate with ROS 2  

## Core Concepts
Isaac Sim bridges simulation and embodied AI deployment through composable scene graphs, ROS plugins, and GPU acceleration.

## Key Takeaways
- Isaac Sim enables physics-aware AI training
